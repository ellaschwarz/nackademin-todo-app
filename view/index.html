      COOKIE POLICY

      What kind of data is saved in cookies?

      In this Todo-application we're saving an identification which is encrypted in a token. We're saving this token in Session Storage in order for you to be able to sign in and to ensure both the user experiece and features of the application

      Why do we save your data in a cookie?

      The data is saved to make sure that you can get authenticated sign in safely. The cookie also helps to improve the user experience when you're using the application

      What is a cookie?

      Cookies are usually small text files, given ID tags that are stored on your computer's browser directory or program data subfolders. Cookies are created when you use your browser to visit a website that uses cookies to keep track of your movements within the site, help you resume where you left off, remember your registered login, theme selection, preferences, and other customization functions.The website stores a corresponding file(with same ID tag)to the one they set in your browser and in this file they can track and keep information on your movements within the site and any information you may have voluntarily given while visiting the website, such as email address.


      INTEGRITY POLICY

      Which data are being saved?

      In this application, we're saving your user information (email and password), which are being provided when you register an account. We're also saving your user data such as todo-lists and todo-item that are created in the application. If you want to collect and see all your personal data that we've saved, just click the 'Get all my data'-button and we'll send you the data to your email.

      Why do we save data and how do we use it?

      The user information is saved in order to identify and authenticate the user, but also to display the right user data to the right user. We're only saving essential data that are necessary in order to make the application function properly

      How do we delete data?

      As a user, you have the right to be forgotten and delete all data that are saved in this application Click on the 'Delete all my data'-button to request your user and user data to be removed. When you click on the button, the user account will be ended and erased as well as all data that belongs to it.

      We use cookies to improve user experience. For these reasons, we may use your usage data to improve the application. By clicking “Accept Cookies,” you consent to store on your device all the data described in our Cookie Policy.
